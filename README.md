# Portal de Facturación - Millennials Tech

## Descripción del Proyecto
Sistema web para la gestión y emisión de facturas automatizadas. Este proyecto busca resolver la carga operativa de **Millennials Tech** mediante un modelo de auto-facturación, permitiendo a los clientes finales generar sus propios comprobantes fiscales de manera sencilla y validada.

## Objetivo del MVP (Fase 1)
Construir un portal funcional que permita:
1. **Login de Clientes:** Acceso seguro para los clientes directos de Millennials Tech.
2. **Generación de Solicitudes:** Creación de pre-facturas.
3. **Auto-facturación:** Formulario público para captura de datos fiscales (RFC, Razón Social, etc.) con validación en tiempo real.
